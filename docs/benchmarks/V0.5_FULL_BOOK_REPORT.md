# v0.5.2 full-book local benchmark

Run date: 2026-07-20

This benchmark exercises the current generic local pipeline on a 125-page Japanese manga while preserving every page's native pixel geometry. Most pages are 2126 × 3661; the final page retains its distinct 2800 × 3808 source geometry. The run used no online LLM API and no book-specific or manually maintained person-name table.

## Runtime configuration

- OCR geometry: PaddleOCR with Manga OCR recognition
- Semantic OCR and context review: `huihui_ai/qwen3.5-abliterated:9b` through local Ollama, used only for missing-text evidence, role auditing, name disambiguation, and wording review
- Independent translation candidate: Hy-MT2 1.8B
- Name consistency: recurring-name extraction, JMnedict lookup, natural Chinese name-candidate filtering, and local 9B disambiguation
- Artwork repair: Big-LaMa with type-size-aware outlined-text halos and bounded edit regions
- Output: native-resolution lossless WebP
- Sound-effect policy: preserve large/decorative source SFX; translate only detector-confirmed ordinary text units
- Geometry authority: detector-owned boxes only; raw VLM boxes never erase or inpaint source pixels

The final output was rendered by the public project CLI from an imported, locally reviewed transcript. Pages 2, 9, 10, 14, 15, and 21 received the targeted 9B semantic/OCR audit before the full render. The three README comparisons are generated from pages 1–3 of this output with `scripts/build_readme_comparisons.py`. The comparison files are resized for GitHub only; every benchmark output keeps its corresponding source page dimensions.

## Verification command

```powershell
uv run python scripts/verify_output.py SOURCE OUTPUT TRANSCRIPT --preserve-sfx --output-format webp
```

The verifier rejects missing pages, zero-byte images, dimension changes, unresolved or kana-bearing Chinese fields, mismatched output formats, and any decoded pixel change outside the declared per-unit edit masks.

## Scope

This is an approximately 85-point quality closure target, not a claim that OCR, literary translation, or LaMa texture reconstruction can be perfect on every drawing style. Some repaired regions can retain soft grayscale texture, and large decorative sound effects are deliberately left in Japanese. The pipeline prefers those visible residuals over erasing clothing prints, props, character lines, or artwork on unconfirmed VLM geometry. All detection, filtering, translation, and rendering rules are based on page and text-unit features; none contain this book's title, dialogue, page numbers, coordinates, or a fixed cast list.
